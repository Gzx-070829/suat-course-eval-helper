# suat-course-eval-helper

## 一键安装

完全不需要会编程，也不需要使用 `git clone`、npm、下载 ZIP 或复制代码。

### 第 1 步：安装 Tampermonkey / 篡改猴

先在浏览器中安装 [Tampermonkey（篡改猴）](https://www.tampermonkey.net/)。安装完成后，再回到本页面选择一个版本。

### 第 2 步：点击所需版本

- **[一键安装 Lite 版](https://raw.githubusercontent.com/Gzx-070829/suat-course-eval-helper/main/userscripts/suat-eval-helper-lite.user.js)**：推荐大多数同学使用；不联网，使用内置文字模板。
- **[一键安装 AI 版](https://raw.githubusercontent.com/Gzx-070829/suat-course-eval-helper/main/userscripts/suat-eval-helper-ai.user.js)**：可选 AI 草稿；只有主动配置自己的 API 后才能使用 AI 功能。

浏览器会打开 Tampermonkey 的脚本安装页面。请点击页面中的“安装”，不要同时安装或启用两个版本。

### 第 3 步：开始使用

1. 打开学校课程评价页面。
2. 点击页面右下角的辅助按钮。
3. 使用辅助填写功能，并逐项检查评分和文字。
4. 确认无误后，在学校页面中**手动保存或提交**。

> 重要：本工具只辅助预填，绝不会自动保存或提交，也不会绕过登录或验证码。最终内容必须由你检查，并由你本人手动保存或提交。

为深圳理工大学（SUAT）/ 原 SIAT 教育信息化平台课程评价页面提供的 Tampermonkey 用户脚本。它可以辅助选择评分并预填开放式评价，减少重复操作，同时始终把最终检查与保存权留给用户。

> 本项目只做表单预填：不会自动保存、提交或确认评价。请逐项检查评价结果，并由你本人手动保存。

## 支持网址

- `https://education.siat.ac.cn/*`
- `https://education.suat-sz.edu.cn/*`

脚本仅在识别到课程评价页面时显示操作界面。

## Lite 版与 AI 版

| 功能 | Lite 版 | AI 版 |
| --- | --- | --- |
| Element UI 评分下拉框预选 | 支持“非常满意” | 可选“非常满意”“满意”或不选择 |
| 开放式评价 | 固定模板 | 固定模板或 AI 草稿 |
| 设置面板 | 无 | 有 |
| 外部网络请求 | 无 | 仅在用户主动生成 AI 草稿时请求自定义 API |
| 自动保存/提交 | 不支持 | 不支持 |

- Lite 版完全不联网，不会向外部服务发送学校页面信息。
- AI 版默认也使用固定模板；AI 功能需要用户主动配置自己的 OpenAI-compatible API，并主动点击生成草稿。

详细步骤见 [使用教程](docs/usage.md)。

## 使用说明

### Lite 版

1. 打开学校课程评价详情页。
2. 点击右下角“辅助填写教评”。
3. 阅读确认框并确认后，脚本会逐项尝试选择“非常满意”，并填写仍为空白的开放式评价。
4. 查看完成统计，逐项检查评分和文字。
5. 确认无误后，由你手动点击学校页面的保存或提交按钮。

### AI 版

1. 打开课程评价详情页，在右下角面板中选择评分和评价模式。
2. 默认使用“固定模板”，此模式不会向外部服务发送页面数据。
3. 如需 AI，主动选择“AI 草稿”，配置 OpenAI-compatible API 并保存设置。
4. 点击“生成草稿”，确认隐私提示。草稿生成后先在面板内修改和确认。
5. 点击“预填页面”，再逐项检查，并手动保存。

AI 配置和数据流说明见 [AI 模式文档](docs/ai-mode.md)。

## 我不会编程，可以用吗？

可以。你只需要会安装浏览器扩展和点击网页按钮，不需要写代码，也不需要安装开发工具。

按照本页顶部的“一键安装”步骤操作即可：先安装 Tampermonkey / 篡改猴，再点击 Lite 版或 AI 版的安装链接，最后在 Tampermonkey 页面点击“安装”。第一次使用建议选择不联网、无需任何配置的 Lite 版。

## 常见问题

### 点安装链接只看到代码怎么办？

这通常表示 Tampermonkey 尚未安装、未启用，或者安装扩展后还没有刷新本页面。请确认浏览器中已经安装并启用 Tampermonkey，刷新本页面，然后再次点击“一键安装”链接。正常情况下会出现 Tampermonkey 的安装页面，再点击其中的“安装”即可；不需要复制看到的代码。

### 打开教评页面后没有辅助按钮怎么办？

请依次检查：Tampermonkey 是否已启用、所选脚本是否已启用、当前是否为学校的课程评价详情页。然后刷新教评页面，并留意页面右下角。脚本只在识别到评价页面时显示按钮。如果仍然没有按钮，可以提交 [Issue](https://github.com/Gzx-070829/suat-course-eval-helper/issues)，但请勿上传姓名、学号、课程、教师、Cookie 或 Token 等隐私信息。

### 是否会自动提交评价？

不会。本工具不会自动点击“保存”“提交”或“确认”，只会辅助预填。请你逐项检查后，亲自在学校页面手动保存或提交。

### AI 版是否会上传信息？

固定模板模式不会发送页面信息。只有当你主动配置自己的 API、选择“AI 草稿”、点击“生成草稿”并确认提示后，AI 版才会把提示中明确展示的信息发送给你配置的 API 服务。项目本身没有数据收集服务器。若不希望产生任何外部请求，请使用 Lite 版。

## 安全边界

本项目有意限制自动化范围：

- 不自动点击“保存”“提交”“确认”等按钮，也不批量提交评价。
- 不绕过登录、验证码、权限控制或学校平台的其他安全机制。
- Lite 版不发起外部请求，不上传学校系统数据。
- AI 版仅在用户主动点击“生成草稿”并再次确认后，将明确展示的信息发送到用户自行配置的 API 服务；项目本身不设数据收集服务器。
- AI 只生成开放式评价草稿，不替用户作最终判断，也不会直接写入页面。
- 所有评分和文字都必须由用户人工检查，最终保存或提交只能由用户手动完成。
- 已有内容的开放式评价框默认不会被覆盖。

设计原则详见 [伦理与使用边界](docs/ethics.md)。
