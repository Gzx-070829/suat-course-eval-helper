# Changelog

本项目的显著变化记录于此。

## [0.5.1] - 2026-07-03

- 修复 Queue 在 Element UI fixed-right 固定操作列中无法匹配“未填写”行与“评价”入口的问题。
- 主表格负责识别提交状态，固定右列按相同 `rowIndex` 配对操作入口。
- 增加列表页扫描诊断日志和表格结构变化提示。
- 保持只点击行内“评价”和详情页“保存”，不会点击最终“提交评价”。

## [0.5.0] - 2026-07-03

- 恢复双脚本结构。
- 新增 Lite 版：当前课程一键预填，不自动保存。
- 新增 Queue 版：列表页队列填写并自动保存当前课程草稿。
- 扩充随机评分和开放式评价模板池。
- 两个版本均不联网、不使用 AI、不收集数据。
- Lite 不自动保存、不自动提交。
- Queue 会自动点击详情页“保存”，但不会点击最终“提交评价”。
- 用户最终仍需人工检查并手动提交评价。

## [0.3.0] - 2026-07-03

### Changed

- 移除复杂面板，页面只保留右下角“教评辅助”按钮。
- 改为点击按钮后直接执行的一键极简预填。
- 增加课程优点和改进建议的本地随机模板组合。
- 增加少量“满意”、大部分“非常满意”的随机评分分布。
- 保持不联网、不自动保存、不自动提交。

## [0.2.0] - 2026-07-03

### Changed

- 移除 AI 版本，项目聚焦单一的本地版教评辅助工具。
- 脚本重命名为 `userscripts/suat-eval-helper.user.js`。
- 更新脚本名称、版本、下载地址和自动更新地址。
- 浮动按钮打开后先显示页面预检，不再立即填写。
- 支持修改优点评价和建议评价两段模板，并保存到浏览器本地存储。
- README 和教程改为单版本一键安装流程。

### Safety

- 保持完全离线，不收集或上传数据。
- 用户确认后才预填。
- 不覆盖已有评分或文字。
- 不自动保存、不自动提交、不绕过登录或验证码。

## [0.1.0] - 2026-07-03

### Added

- 新增本地用户脚本，支持识别 SIAT/SUAT 课程评价页。
- 新增评分下拉框辅助选择和开放式评价模板预填。
- 新增 SPA 页面切换支持。
- 明确禁止自动保存、提交、确认和批量评价。
