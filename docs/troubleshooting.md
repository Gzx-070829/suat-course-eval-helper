# 常见问题

## 点击安装链接只看到代码怎么办？

请先安装并启用 Tampermonkey，刷新 README 页面后再次点击安装链接。正常情况下会打开 Tampermonkey 安装页面，不需要复制代码。

## 页面没有按钮怎么办？

确认 Tampermonkey 和所选脚本均已启用。Lite 只在课程评价详情页显示按钮；Queue 在评价课程列表页和详情页显示按钮。等待页面加载完成或刷新后重试。

## 同时启用两个脚本怎么办？

打开 Tampermonkey 管理面板，停用其中一个。推荐普通用户保留 Lite；需要列表队列时再改用 Queue。

## Lite 填写失败怎么办？

页面可能尚未加载完成、课程已经不可编辑，或者学校更新了页面结构。请刷新后重试，并手动完成脚本未填写的项目。

## Queue 队列卡住怎么办？

按住 `Alt` 点击“教评辅助”清除队列状态，然后检查当前课程和列表状态。不要在不清楚当前保存情况时反复启动队列。

## Queue 自动保存失败怎么办？

Queue 无法找到唯一的“保存”按钮，或没有检测到“保存成功”提示时会停止。请手动检查当前课程、手动保存，再返回列表确认状态。

## 如何清除 Queue 状态？

在 Queue 按钮可见时，按住 `Alt` 并点击“教评辅助”。看到“Queue 队列状态已清除”后即可重新开始。

## 如何停用 Queue 改用 Lite？

先清除 Queue 状态，再到 Tampermonkey 管理面板停用 Queue、安装或启用 Lite，最后刷新学校页面。

## 如何反馈问题？

前往 [GitHub Issues](https://github.com/Gzx-070829/suat-course-eval-helper/issues)，说明浏览器版本、Tampermonkey 版本、使用的是 Lite 还是 Queue，以及失败步骤。

请勿上传姓名、学号、课程、教师、评价内容、Cookie、Token 或完整页面截图。
