# 常见问题与排障

## 点击安装链接只看到代码怎么办？

通常是 Tampermonkey 尚未安装、未启用，或者安装扩展后没有刷新页面。

1. 确认浏览器中已经安装并启用 Tampermonkey / 篡改猴。
2. 刷新项目 README 页面。
3. 再次点击“一键安装”。
4. 正常情况下会进入 Tampermonkey 安装页面，再点击“安装”。

不需要复制看到的代码。

## 页面没有按钮怎么办？

请依次检查：

1. Tampermonkey 是否已启用。
2. “SIAT/SUAT 教师评价辅助填写器”是否已启用。
3. 当前是否为某门课程的评价详情页，而不是教务首页或课程列表。
4. 页面是否仍在加载，可以等待片刻或刷新后重试。
5. 当前网址是否属于支持的两个学校域名。

脚本只在疑似教评详情页显示按钮。

## 按钮出现但没有填成功怎么办？

先查看面板中的预检结果和填写完成统计。常见原因包括：

- 当前项目已经有内容，脚本为避免覆盖而跳过。
- 评分选项文字或页面结构发生变化。
- 下拉框无法被唯一、安全地识别。
- 当前课程已经评价完成，字段不可编辑。
- 页面在预填过程中发生了切换。

可以点击“查看诊断信息”，复制脱敏后的诊断结果用于反馈。请先手动完成未填写的项目，不要反复强制运行。

## 如何卸载脚本？

打开 Tampermonkey 管理面板，找到“SIAT/SUAT 教师评价辅助填写器”，点击删除。卸载后刷新学校页面，右下角按钮就会消失。

## 如何反馈问题？

前往 [GitHub Issues](https://github.com/Gzx-070829/suat-course-eval-helper/issues) 提交问题，并说明：

- 浏览器名称和版本
- Tampermonkey 版本
- 问题发生在哪个操作步骤
- 面板中复制的诊断信息

请勿上传姓名、学号、课程、教师、评价内容、Cookie、Token、页面完整截图或其他敏感信息。
