# 常见问题

## 页面没有按钮怎么办？

请确认 Tampermonkey 和脚本均已启用，并且当前已经进入某门课程的评价详情页。等待页面加载完成或刷新后重试。脚本不会在普通教务首页和课程列表页显示按钮。

## 点击安装链接只看到代码怎么办？

请先安装并启用 Tampermonkey，刷新 README 页面后再次点击一键安装链接。正常情况下会打开 Tampermonkey 安装页面，不需要复制代码。

## 填写失败怎么办？

可能原因包括页面尚未加载完成、课程已经评价、控件不可编辑，或者学校更新了页面结构。请先手动完成未填写项目，并在提交前检查所有评分和文字。

如需反馈，请前往 [GitHub Issues](https://github.com/Gzx-070829/suat-course-eval-helper/issues)。请勿上传姓名、学号、课程、教师、评价内容、Cookie、Token 或完整页面截图。

## 如何卸载？

打开 Tampermonkey 管理面板，找到“SIAT/SUAT 教师评价辅助填写器”，点击删除，然后刷新学校页面。
