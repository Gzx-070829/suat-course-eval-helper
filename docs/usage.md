# 使用教程

本项目只有一个纯本地版本：不联网、不收集数据，也不会自动保存或提交。

## 1. 安装 Tampermonkey

打开 [Tampermonkey 官网](https://www.tampermonkey.net/)，选择自己的浏览器并安装“Tampermonkey / 篡改猴”。安装完成后，确认浏览器工具栏中可以看到它的图标。

## 2. 一键安装脚本

点击 [安装教评辅助填写器](https://raw.githubusercontent.com/Gzx-070829/suat-course-eval-helper/main/userscripts/suat-eval-helper.user.js)，然后在 Tampermonkey 安装页面点击“安装”。

不需要 GitHub 账号，不需要命令行，不需要 npm，也不需要复制代码。

## 3. 打开学校教评页面

按正常方式登录学校教育信息化平台，并进入某门课程的评价详情页。脚本支持：

- `https://education.siat.ac.cn/*`
- `https://education.suat-sz.edu.cn/*`

脚本不能绕过登录、验证码或权限控制。

## 4. 打开预检面板

点击页面右下角“教评辅助”。此时脚本不会立即填写，而是先显示：

- 是否识别为教评页面
- 发现的评分下拉框数量
- 发现的文本框数量
- 准备选择的评分
- 准备使用的优点评价和建议评价模板

## 5. 修改设置和模板

评分可以选择：

- 非常满意（默认）
- 满意
- 不自动选择

请根据自己的真实体验修改“优点评价模板”和“建议评价模板”。点击“保存设置”后，评分和模板会保存在当前浏览器的本地存储中。

## 6. 检查逐题预览

脚本会根据题目附近文字选择较合适的模板，并显示逐题预览。你可以继续修改，也可以勾选“本题不填写”。已有内容固定跳过。

## 7. 确认并预填

点击“确认并预填”，再次阅读预检统计并确认。只有确认后脚本才会开始处理，而且会在写入前再次检查字段是否为空。

## 8. 检查并手动保存

预填完成后，请上下检查学校页面：

1. 评分是否符合自己的真实感受。
2. 每段文字是否回答了对应题目。
3. 是否存在不准确或不想提交的表达。
4. 是否有因安全原因被跳过的项目。

确认无误后，由你本人手动保存或提交。脚本永远不会代替你完成这一步。

## 9. 撤销本次填写

点击“撤销本次填写”，可以撤销最近一次由脚本写入的内容。如果你在预填后手动修改了某个字段，撤销会跳过它，避免覆盖你的新内容。

部分评分框可能没有可靠的清空能力。遇到这种情况，脚本会提醒你手动检查，不会执行不确定的操作。

## 10. 卸载或更新

- 卸载：打开 Tampermonkey 管理面板，找到“SIAT/SUAT 教师评价辅助填写器”，点击删除。
- 更新：Tampermonkey 会按脚本内置地址检查更新；也可以重新点击一键安装链接。

更多帮助见 [常见问题与排障](troubleshooting.md)。
