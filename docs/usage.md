# Lite 版使用教程

Lite 版适合绝大多数用户：完全离线、不需要 API，也不会自动保存或提交。

## 1. 安装 Tampermonkey

打开 [Tampermonkey 官网](https://www.tampermonkey.net/)，选择自己的浏览器并安装“Tampermonkey / 篡改猴”扩展。安装后确认浏览器工具栏中可以看到它的图标。

## 2. 一键安装 Lite 版

点击 [安装 Lite 版](https://raw.githubusercontent.com/Gzx-070829/suat-course-eval-helper/main/userscripts/suat-eval-helper-lite.user.js)。浏览器会打开 Tampermonkey 安装页面，再点击其中的“安装”。

不需要下载 ZIP、复制代码、安装 npm 或使用命令行。

## 3. 打开学校教评页面

按正常方式登录学校教育信息化平台，进入某门课程的评价详情页：

- `https://education.siat.ac.cn/teaching/evaluation`
- `https://education.suat-sz.edu.cn/teaching/evaluation`

脚本不能绕过登录、验证码或权限控制，也不会在普通首页和课程列表页随意显示。

## 4. 展开辅助面板

进入评价详情页后，点击右下角“教评辅助”。选择：

- 评分：非常满意、满意或不填写评分。
- 文本风格：自然正式、简洁或具体建设性。

脚本固定只处理空白项，不提供覆盖开关。

## 5. 扫描并生成草稿

点击“扫描并生成草稿”。Lite 会在本地读取开放题附近的题目文字，将问题识别为优点、建议、互动、实践、内容或通用类型，并生成相应草稿。

整个过程不联网。

## 6. 修改草稿

每个开放题都会显示题目、识别类型和可编辑草稿。请根据自己的真实体验修改。若不想填写某题，勾选“本题不填写”。已有内容会自动跳过。

## 7. 采用并预填

点击“采用并预填”，阅读提示并确认。脚本会再次检查字段是否仍为空，只把你确认过的草稿写入空白文本框，并安全尝试填写空白评分项。

## 8. 上下检查页面

预填完成后，请上下滚动学校页面，检查：

1. 每个评分是否符合真实感受。
2. 每段文字是否回答了对应问题。
3. 是否存在不准确、模板化或不想提交的表达。
4. 是否还有脚本因安全原因跳过的项目。

## 9. 手动保存

确认无误后，由你本人点击学校页面上的保存或提交。脚本永远不会代替你执行这一步。

## 10. 撤销本次填写

点击“撤销本次填写”可以撤销最近一次由脚本写入的内容。若你在预填后修改过某个字段，撤销会跳过它，以免覆盖你的新内容。

部分 Element UI 评分框可能没有可靠的清空能力。遇到这种情况，脚本会保留评分并提示手动检查，不会尝试不确定的操作。

## 11. 查看诊断信息

点击“查看诊断信息”可以查看页面识别结果和控件数量。诊断信息不包含课程名、教师名、评价内容或用户草稿，可以复制后用于提交脱敏 Issue。

## 12. 卸载脚本

点击浏览器工具栏中的 Tampermonkey 图标，进入管理面板，找到“SIAT/SUAT 教师评价辅助填写 Lite”，点击删除。

## 13. 更新脚本

Tampermonkey 会根据脚本内置的更新地址检查新版。也可以重新点击一键安装链接，并在 Tampermonkey 页面确认更新。
